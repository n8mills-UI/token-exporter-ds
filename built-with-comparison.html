Cons

  • Takes more vertical space
  • Less visual interest
  • More scrolling required
  • Less compact

Recommendation

For mobile-first design: The 2x4 column layout provides a good balance between space efficiency and usability. It saves vertical space while maintaining adequate touch targets and readability.

For desktop: The 4x2 grid layout is more visually compact and efficient for larger screens where space is less constrained.

Hybrid approach: Consider using the horizontal layout on mobile (< 768px) and the grid layout on larger screens for the best of both worlds.
