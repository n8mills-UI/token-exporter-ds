Button Usage Guidelines

  • Primary: Main actions (Save, Submit, Get Started)
  • Secondary: Alternative actions (Cancel, Learn More)
  • Tertiary: Low-emphasis actions (Skip, Maybe Later)
  • Icon-Only: Social links, compact toolbars, close buttons
  • Sizing: Use md as default, sm for inline/compact, lg for hero CTAs
  • Icons: Always use <span class="icon" data-icon="name"></span>
