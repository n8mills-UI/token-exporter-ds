7. Gradient Directional Variants - Remove Redundant Text & Reduce Height

❌ Current (Too tall, redundant text)

45°
--gradient-hero-1-45 45° variant

Problems:
• White box is visually distracting
• Square aspect ratio forces tall cards
• "45° variant" text is redundant
• Excessive padding all around

✅ Proposed (Compact, no redundant text)

45°
--gradient-hero-1-45

Fixes:
• No white box - subtle text shadow instead
• Centered angle text on gradient
• Fixed height (60px) instead of square
• No redundant degree text
• Reduced padding to var(--size-2)

Implementation Requirements:
• CSS: Add class for directional variants with reduced padding and fixed height
• JavaScript: Detect tokens ending in -45, -90, -135, -180 and don't add hex span
• Result: Compact cards that match the density of other card types
