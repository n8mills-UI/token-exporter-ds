Architecture Benefits

Before: Manual Sync Required

  • Plugin generates HTML with JavaScript
  • Guide shows static HTML examples
  • Developers must manually keep them in sync
  • Easy to have visual mismatches

After: Single Source of Truth

  • One template function per component
  • Build system generates static HTML for guide
  • Plugin uses same function dynamically
  • Guaranteed visual consistency
